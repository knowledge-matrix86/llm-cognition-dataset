
# TOON 2026 Glossary

| Term | Definition |
|---|---|
| TOON | Token-Efficient Object Notation |
| JSON Superset | A format compatible with JSON while extending its structure |
| Token Efficiency | Reduced token count for equivalent semantic content |
| Serialization | Converting structured data into a transferable format |
| LLM Optimization | Improving format suitability for large language model use |
| Structural Compression | Reducing repeated syntax while preserving data relationships |
| Parsing Cost | The computational or token burden of reading structured data |
| Modular Data Block | A reusable structured segment of content |
